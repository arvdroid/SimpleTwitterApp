This is Twitter App to get users tweets

Time Spent: 27 hours

Completed User stories:

1. User can sign in to Twitter using OAuth login
2. User can view the tweets from their home timeline
	User should be able to see the username, name, body and timestamp for each tweet
	User should be displayed the relative timestamp for a tweet "8m", "7h"
	User can view more tweets as they scroll with infinite pagination.
3. User can compose a new tweet
	User can click a “Compose” icon in the Action Bar on the top right
	User can then enter a new tweet and post this to twitter
	User is taken back to home timeline with new tweet visible in timeline

Optional:

1. User can refresh tweets timeline by pulling down to refresh (i.e pull-to-refresh)

2. User can open the twitter app offline and see last loaded tweets

3. User can tap a tweet to display a "detailed" view of that tweet

4. User select "reply" from detail view to respond to a tweet

5. User can delete tweet from detail view.
